## General Information

Rich text editor project, created with next js + slate js + antd + fontawesome + typescript. Only basic feature implemented with some custom component, still need to be optimised and do some code refactoring to make it more cleaner.

## Getting Started

First, install the node modules:

```bash
pnpm install 
```

Then, run the dev server by using:
```bash
pnpm run dev 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
